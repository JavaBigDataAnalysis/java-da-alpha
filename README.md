# JavaBigDataAnalysis
### Java下的大数据分析程序-分析模块

* 超大数据文件的读写
* 基于正则表达式的字段提取
* 基于红黑树和哈希散列技术的存储搜索结构
* 基于Stream流的并行计算
* 可确保线程安全的并行计算
